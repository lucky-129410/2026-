## 多维度分类（Multi-Output Classification）

**多维度分类**是指一个模型同时进行多个独立的分类任务，每个任务都有自己的输出和损失函数。

**典型应用场景**：
- **人物属性识别**：同时预测 性别（男/女）、年龄段（青少年/成年/老年）、眼镜（有/无）
- **商品分类**：同时预测 主类别（电子/服装/食品）、子类别、品牌、颜色
- **医学诊断**：同时检测是否患有 糖尿病、高血压、心脏病等多种病症
- **图像场景理解**：同时识别 室内/室外、白天/夜晚、天气状况

**与相似概念的区别**：

| 概念 | 定义 | 标签格式 | 损失函数 |
|------|------|--------|---------|
| 二分类 | 1个任务，2个类别 | [0/1] | BCELoss |
| 多分类 | 1个任务，≥3个类别 | [0-9] | CrossEntropyLoss |
| 多标签分类 | 1个任务，多个非互斥标签 | [1,1,0,1] | BCELoss |
| 多维度分类 | 多个任务，每个任务独立 | [性别, 年龄, 眼镜] | 多个损失函数之和 |

---

## 网络结构

**通用的多维度分类网络架构**：

```
输入数据
   ↓
━━━━━━ 共享主干网络（Backbone） ━━━━
   ↓
分支1 分支2 分支3 ... 分支n
   ↓      ↓      ↓         ↓
输出1  输出2  输出3 ... 输出n
```

**关键特点**：
- **共享主干**：所有任务共用底层特征提取器（节省参数）
- **独立分支**：每个任务有自己的分类头（输出层）
- **独立损失**：每个任务的预测与标签独立计算损失

## PyTorch 实现

### 1. 简单的多维度分类网络

```python
import torch
import torch.nn as nn

class MultiTaskClassifier(nn.Module):
    """多维度分类模型"""
    def __init__(self, input_size=784, hidden_size=128):
        super().__init__()
        
        # 共享主干网络
        self.backbone = nn.Sequential(
            nn.Linear(input_size, hidden_size),
            nn.ReLU(),
            nn.Linear(hidden_size, hidden_size),
            nn.ReLU()
        )
        
        # 任务1的分类头（例如：性别分类，2个类别）
        self.task1_head = nn.Linear(hidden_size, 2)
        
        # 任务2的分类头（例如：年龄段分类，3个类别）
        self.task2_head = nn.Linear(hidden_size, 3)
        
        # 任务3的分类头（例如：眼镜分类，2个类别）
        self.task3_head = nn.Linear(hidden_size, 2)
    
    def forward(self, x):
        # 共享特征提取
        shared_features = self.backbone(x)
        
        # 各任务的独立预测
        task1_logits = self.task1_head(shared_features)
        task2_logits = self.task2_head(shared_features)
        task3_logits = self.task3_head(shared_features)
        
        return task1_logits, task2_logits, task3_logits


# 创建模型
model = MultiTaskClassifier(input_size=784, hidden_size=128)

# 模拟数据
batch_size = 32
x = torch.randn(batch_size, 784)

# 前向传播
task1_out, task2_out, task3_out = model(x)

print(f"任务1输出形状: {task1_out.shape}")  # [32, 2]
print(f"任务2输出形状: {task2_out.shape}")  # [32, 3]
print(f"任务3输出形状: {task3_out.shape}")  # [32, 2]
```

### 2. 训练多维度分类模型

```python
import torch
import torch.nn as nn
import torch.optim as optim

class MultiTaskClassifier(nn.Module):
    def __init__(self, input_size=784, hidden_size=128):
        super().__init__()
        self.backbone = nn.Sequential(
            nn.Linear(input_size, hidden_size),
            nn.ReLU(),
            nn.Linear(hidden_size, hidden_size),
            nn.ReLU()
        )
        self.task1_head = nn.Linear(hidden_size, 2)   # 性别：男/女
        self.task2_head = nn.Linear(hidden_size, 3)   # 年龄：少/中/老
        self.task3_head = nn.Linear(hidden_size, 2)   # 眼镜：有/无
    
    def forward(self, x):
        features = self.backbone(x)
        return self.task1_head(features), self.task2_head(features), self.task3_head(features)

# 模型和优化器
model = MultiTaskClassifier()
optimizer = optim.Adam(model.parameters(), lr=0.001)

# 损失函数
criterion1 = nn.CrossEntropyLoss()  # 任务1的交叉熵损失
criterion2 = nn.CrossEntropyLoss()  # 任务2的交叉熵损失
criterion3 = nn.CrossEntropyLoss()  # 任务3的交叉熵损失

# 模拟批次数据
x_batch = torch.randn(32, 784)
y_task1 = torch.randint(0, 2, (32,))  # 性别标签：0或1
y_task2 = torch.randint(0, 3, (32,))  # 年龄标签：0、1或2
y_task3 = torch.randint(0, 2, (32,))  # 眼镜标签：0或1

def train_step():
    # 前向传播
    pred1, pred2, pred3 = model(x_batch)
    
    # 计算每个任务的损失
    loss1 = criterion1(pred1, y_task1)
    loss2 = criterion2(pred2, y_task2)
    loss3 = criterion3(pred3, y_task3)
    
    # 总损失 = 所有任务损失之和（可以加权重）
    total_loss = loss1 + loss2 + loss3
    
    # 反向传播和优化
    optimizer.zero_grad()
    total_loss.backward()
    optimizer.step()
    
    return loss1.item(), loss2.item(), loss3.item(), total_loss.item()

# 训练
loss1, loss2, loss3, total = train_step()
print(f"任务1损失: {loss1:.4f}")
print(f"任务2损失: {loss2:.4f}")
print(f"任务3损失: {loss3:.4f}")
print(f"总损失: {total:.4f}")
```

### 3. 加权多维度分类（重要任务权重更大）

```python
import torch
import torch.nn as nn

# 定义损失权重
TASK_WEIGHTS = {
    'task1': 1.0,      # 性别分类权重
    'task2': 1.5,      # 年龄分类权重更高
    'task3': 0.5       # 眼镜分类权重较低
}

def compute_weighted_loss(pred1, pred2, pred3, y1, y2, y3):
    """计算加权多任务损失"""
    criterion = nn.CrossEntropyLoss()
    
    loss1 = criterion(pred1, y1) * TASK_WEIGHTS['task1']
    loss2 = criterion(pred2, y2) * TASK_WEIGHTS['task2']
    loss3 = criterion(pred3, y3) * TASK_WEIGHTS['task3']
    
    total_loss = loss1 + loss2 + loss3
    
    return total_loss, loss1, loss2, loss3
```

### 4. 推理和评估

```python
import torch
import torch.nn as nn

def evaluate_multi_task(model, x_test, y_test1, y_test2, y_test3):
    """评估多维度分类模型"""
    model.eval()
    
    with torch.no_grad():
        pred1, pred2, pred3 = model(x_test)
        
        # 获得预测类别
        pred1_classes = torch.argmax(pred1, dim=1)
        pred2_classes = torch.argmax(pred2, dim=1)
        pred3_classes = torch.argmax(pred3, dim=1)
        
        # 计算准确率
        acc1 = (pred1_classes == y_test1).float().mean().item()
        acc2 = (pred2_classes == y_test2).float().mean().item()
        acc3 = (pred3_classes == y_test3).float().mean().item()
        
        return {
            'task1_accuracy': acc1,
            'task2_accuracy': acc2,
            'task3_accuracy': acc3,
            'average_accuracy': (acc1 + acc2 + acc3) / 3
        }

# 使用示例
x_test = torch.randn(1000, 784)
y_test1 = torch.randint(0, 2, (1000,))
y_test2 = torch.randint(0, 3, (1000,))
y_test3 = torch.randint(0, 2, (1000,))

metrics = evaluate_multi_task(model, x_test, y_test1, y_test2, y_test3)
print(f"任务1准确率: {metrics['task1_accuracy']:.4f}")
print(f"任务2准确率: {metrics['task2_accuracy']:.4f}")
print(f"任务3准确率: {metrics['task3_accuracy']:.4f}")
print(f"平均准确率: {metrics['average_accuracy']:.4f}")
```

## 多维度分类的优势与挑战

**优势**：
- ✓ **参数共享**：多个任务共用底层特征，减少参数数量
- ✓ **泛化性能好**：共享表示学习到更通用的特征
- ✓ **学习效率高**：一次前向传播完成多个预测任务
- ✓ **正则化效果**：多个任务相当于隐式正则化

**挑战**：
- ✗ **任务冲突**：不同任务的梯度可能冲突，导致学习困难
- ✗ **权重平衡**：需要合理调整各任务的损失权重
- ✗ **模型复杂性**：需要为每个任务设计合适的输出头
- ✗ **不平衡数据**：不同任务的数据量或难度差异大

**改进策略**：
1. **动态权重调整**：根据损失大小动态调整任务权重
2. **梯度归一化**：使用 gradient normalization 平衡任务梯度
3. **任务相关性建模**：考虑任务之间的关系
4. **注意力机制**：使用注意力为不同任务分配资源

## 完整的多维度分类示例

```python
import torch
import torch.nn as nn
import torch.optim as optim
from torch.utils.data import DataLoader, TensorDataset

# 数据生成
def create_synthetic_data(num_samples=1000):
    """生成合成的多维度分类数据"""
    x_data = torch.randn(num_samples, 784)
    
    # 生成不相关的标签（模拟真实场景）
    y_task1 = torch.randint(0, 2, (num_samples,))    # 任务1：2类
    y_task2 = torch.randint(0, 3, (num_samples,))    # 任务2：3类
    y_task3 = torch.randint(0, 2, (num_samples,))    # 任务3：2类
    
    return x_data, y_task1, y_task2, y_task3

# 构建数据加载器
x_train, y1_train, y2_train, y3_train = create_synthetic_data(800)
x_test, y1_test, y2_test, y3_test = create_synthetic_data(200)

train_dataset = TensorDataset(x_train, y1_train, y2_train, y3_train)
test_dataset = TensorDataset(x_test, y1_test, y2_test, y3_test)

train_loader = DataLoader(train_dataset, batch_size=32, shuffle=True)
test_loader = DataLoader(test_dataset, batch_size=32, shuffle=False)

# 定义模型
class MultiTaskClassifier(nn.Module):
    def __init__(self):
        super().__init__()
        self.backbone = nn.Sequential(
            nn.Linear(784, 256),
            nn.ReLU(),
            nn.Dropout(0.5),
            nn.Linear(256, 128),
            nn.ReLU(),
            nn.Dropout(0.3)
        )
        self.task1_head = nn.Linear(128, 2)
        self.task2_head = nn.Linear(128, 3)
        self.task3_head = nn.Linear(128, 2)
    
    def forward(self, x):
        features = self.backbone(x)
        return (self.task1_head(features), 
                self.task2_head(features), 
                self.task3_head(features))

# 初始化
model = MultiTaskClassifier()
optimizer = optim.Adam(model.parameters(), lr=0.001)
criterion = nn.CrossEntropyLoss()
epochs = 10

# 训练循环
print("开始训练...")
for epoch in range(epochs):
    model.train()
    total_loss = 0
    
    for x_batch, y1_batch, y2_batch, y3_batch in train_loader:
        # 前向传播
        pred1, pred2, pred3 = model(x_batch)
        
        # 计算损失
        loss1 = criterion(pred1, y1_batch)
        loss2 = criterion(pred2, y2_batch)
        loss3 = criterion(pred3, y3_batch)
        total_loss_batch = loss1 + loss2 + loss3
        
        # 反向传播
        optimizer.zero_grad()
        total_loss_batch.backward()
        optimizer.step()
        
        total_loss += total_loss_batch.item()
    
    # 评估
    model.eval()
    with torch.no_grad():
        accuracies = [0, 0, 0]
        for x_batch, y1_batch, y2_batch, y3_batch in test_loader:
            pred1, pred2, pred3 = model(x_batch)
            accuracies[0] += (torch.argmax(pred1, dim=1) == y1_batch).float().mean()
            accuracies[1] += (torch.argmax(pred2, dim=1) == y2_batch).float().mean()
            accuracies[2] += (torch.argmax(pred3, dim=1) == y3_batch).float().mean()
        
        num_batches = len(test_loader)
        avg_loss = total_loss / len(train_loader)
        print(f"Epoch {epoch+1}/{epochs} | Loss: {avg_loss:.4f} | "
              f"Acc1: {accuracies[0]/num_batches:.4f}, "
              f"Acc2: {accuracies[1]/num_batches:.4f}, "
              f"Acc3: {accuracies[2]/num_batches:.4f}")

print("训练完成！")
```

## 总结

**多维度分类的关键要点**：
1. **网络结构**：共享主干 + 独立分类头
2. **损失计算**：多个任务的损失函数相加（可能带权重）
3. **权重平衡**：需要调整各任务的相对重要性
4. **评估指标**：分别评估每个任务的性能
5. **实际应用**：广泛应用于属性识别、医学诊断、场景理解等问题
