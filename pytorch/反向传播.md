### 反向传播算法(BP)-->计算出最后的误差对前面某一权重的偏导数

反向传播是计算神经网络梯度的算法，通过链式求导法则从输出层向输入层逐层计算梯度。

**核心思想**：
- 计算神经网络中每个参数（权重和偏置）对最终误差的"贡献度"
- 指导如何修改这些参数，从而减小误差
- 通过梯度下降优化参数

**反向传播过程**：
1. **前向传播**：输入 → 隐层 → 输出，计算预测值
2. **计算损失**：比较预测值与真实值，得到损失函数值
3. **反向传播**：从输出层开始，逐层向后计算梯度
4. **参数更新**：使用学习率和梯度更新权重 w = w - lr × ∂L/∂w

**链式求导法则（Chain Rule）**：
$$\frac{\partial L}{\partial w} = \frac{\partial L}{\partial a} \times \frac{\partial a}{\partial z} \times \frac{\partial z}{\partial w}$$

其中：
- L：损失函数
- a：激活函数输出
- z：加权和（线性层输出）
- w：权重

**梯度计算示例**：
```
假设神经网络：输入 → 线性层 → ReLU → 输出 → 损失
设 x为输入，w为权重，b为偏置，y为真实标签

前向：z = x×w + b  →  a = ReLU(z)  →  L = (a - y)²

反向：
∂L/∂a = 2(a - y)
∂L/∂z = ∂L/∂a × ∂a/∂z  （ReLU导数：z>0时为1，否则为0）
∂L/∂w = ∂L/∂z × ∂z/∂w = ∂L/∂z × x
∂L/∂b = ∂L/∂z × ∂z/∂b = ∂L/∂z × 1
```

### Tensor

**Tensor 是什么**：
- PyTorch 中的基本数据结构，多维数组的抽象概念
- 标量（0阶）、向量（1阶）、矩阵（2阶）、高阶张量等

**Tensor 的维度**：
- **0D Tensor**：标量，shape = ()，例如 5
- **1D Tensor**：向量，shape = (n,)，例如 [1,2,3]
- **2D Tensor**：矩阵，shape = (m,n)，例如 [[1,2],[3,4]]
- **3D Tensor**：shape = (d1,d2,d3)，例如图像 (C,H,W)
- **4D Tensor**：批次图像 (batch_size,channels,height,width)

**Tensor 的关键属性**：
- `data`：存储的数据
- `grad`：梯度（反向传播计算得到）
- `requires_grad`：是否需要计算梯度（默认False）
- `grad_fn`：记录该 Tensor 是如何生成的（用于反向传播）

**Tensor 在反向传播中的作用**：
```python
import torch

# 创建需要梯度的 Tensor
x = torch.tensor([2.0], requires_grad=True)
w = torch.tensor([3.0], requires_grad=True)

# 前向传播
y = x * w  # y = 6

# 计算损失
loss = y ** 2  # loss = 36

# 反向传播
loss.backward()

# 查看梯度
print(x.grad)  # ∂loss/∂x
print(w.grad)  # ∂loss/∂w
```

**常用 Tensor 操作**：
- 创建：`torch.tensor()`, `torch.zeros()`, `torch.ones()`, `torch.randn()`
- 运算：加减乘除、矩阵乘法 `@` 或 `torch.mm()`
- 形状变换：`.reshape()`, `.view()`, `.transpose()`
- 梯度控制：`.requires_grad=True`, `.detach()`, `with torch.no_grad():`
