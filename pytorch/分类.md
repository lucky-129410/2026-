## 分类问题基础

**分类**是指将输入数据根据其特征分配到离散的类别中。

**核心任务**：
- 根据输入 $x$，找出其所属于每个类别的**概率**
- 从概率值中找出**最大值**对应的类别
- $y \in \{C_1, C_2, ..., C_k\}$（离散型集合）

**分类 vs 回归**：

| 任务 | 输出 | 例子 | 函数 |
|------|------|------|------|
| 分类 | 离散：类别标签 | 考试通过/不通过、猫/狗/鸟 | Sigmoid、SoftMax |
| 回归 | 连续：具体数值 | 房价、温度、收入 | 线性函数 |

**分类的应用场景**：
- **医学诊断**：患病（1）/ 健康（0）
- **考试结果**：通过（1）/ 未通过（0）
- **图像分类**：猫 / 狗 / 鸟 / 其他
- **垃圾邮件检测**：垃圾（1）/ 正常（0）

## Sigmoid 函数

**Sigmoid 函数**的作用：将任意实数映射到 **[0, 1]** 区间，用于二分类的概率输出

**公式**：
$$\sigma(z) = \frac{1}{1 + e^{-z}}$$

**性质**：
- 输入范围：$(-\infty, +\infty)$
- 输出范围：$(0, 1)$
- 可微分：支持反向传播
- $z = 0$ 时，$\sigma(0) = 0.5$

**Sigmoid 曲线**：
```
      y
      1 |        -------
        |       /
    0.5 |------/
        |     /
      0 |___/________
        +-----------z
       -∝  0  +∝
```

**Sigmoid 的表现**：
- 当 $z \to +\infty$ 时，$\sigma(z) \to 1$
- 当 $z \to -\infty$ 时，$\sigma(z) \to 0$
- 当 $z = 0$ 时，$\sigma(z) = 0.5$

**Sigmoid 和其导数的关系**：
$$\sigma'(z) = \sigma(z) \times (1 - \sigma(z))$$

**PyTorch 中使用 Sigmoid**：

```python
import torch
import torch.nn as nn

# 方法1：直接使用函数
sigmoid = nn.Sigmoid()
z = torch.tensor([-2.0, -1.0, 0.0, 1.0, 2.0])
y = sigmoid(z)
print(y)  # tensor([0.1192, 0.2689, 0.5000, 0.7311, 0.8808])

# 方法2：在层中使用
class BinaryClassifier(nn.Module):
    def __init__(self, input_size):
        super().__init__()
        self.fc = nn.Linear(input_size, 1)
        self.sigmoid = nn.Sigmoid()
    
    def forward(self, x):
        logits = self.fc(x)
        probs = self.sigmoid(logits)
        return probs

# 方法3：使用 functional API
import torch.nn.functional as F
probs = torch.sigmoid(z)
```

## 二分类损失函数

### 交叉熵损失（BCE Loss）

**BCE（Binary Cross Entropy）** 是二分类的标准损失函数

**公式**：
$$L = -\frac{1}{n}\sum_{i=1}^{n} [y_i \log(\hat{y}_i) + (1-y_i)\log(1-\hat{y}_i)]$$

其中：
- $y_i$：真实标签（0 或 1）
- $\hat{y}_i$：模型预测的正类概率（0 到 1 之间）
- $n$：样本数量

**损失直观理解**：

```
当 y=1（正类）时：
  L = -log(y_pred)
  如果 y_pred=0.9，损失低（预测正确）
  如果 y_pred=0.1，损失高（预测错误）

当 y=0（负类）时：
  L = -log(1 - y_pred)
  如果 y_pred=0.1，损失低（预测正确）
  如果 y_pred=0.9，损失高（预测错误）
```

**计算示例**：

```
批次：[y_1=1, y_2=0, y_3=1]
预测：[y_pred_1=0.8, y_pred_2=0.3, y_pred_3=0.6]

单个样本的损失：
L_1 = -(1 × log(0.8) + 0 × log(0.2)) = -log(0.8) ≈ 0.223
L_2 = -(0 × log(0.3) + 1 × log(0.7)) = -log(0.7) ≈ 0.357
L_3 = -(1 × log(0.6) + 0 × log(0.4)) = -log(0.6) ≈ 0.511

平均损失 = (0.223 + 0.357 + 0.511) / 3 ≈ 0.364
```

**PyTorch 中使用 BCE Loss**：

```python
import torch
import torch.nn as nn

# 方法1：BCELoss（需要手动应用 Sigmoid）
predictions = torch.tensor([[0.8], [0.3], [0.6]])
targets = torch.tensor([[1.0], [0.0], [1.0]])

loss_fn = nn.BCELoss()
loss = loss_fn(predictions, targets)
print(f"损失: {loss:.4f}")

# 方法2：BCEWithLogitsLoss（推荐！自动应用 Sigmoid）
logits = torch.tensor([[-0.2], [-1.2], [-0.4]])
targets = torch.tensor([[1.0], [0.0], [1.0]])

loss_fn = nn.BCEWithLogitsLoss()
loss = loss_fn(logits, targets)  # 内部自动 Sigmoid
print(f"损失: {loss:.4f}")
```

## 决策门槛（Decision Threshold）

**决策门槛**是将概率转换为类别标签的临界值

**标准决策规则**：
- 若 $\hat{y} \geq 0.5$，预测为正类（1）
- 若 $\hat{y} < 0.5$，预测为负类（0）

**改变决策门槛的影响**：

```
当门槛=0.5时（默认）：
模型输出：0.3  0.5  0.7
预测标签：0    1    1

当门槛=0.7时（更严格）：
模型输出：0.3  0.5  0.7
预测标签：0    0    1
          （对正类更保守）

当门槛=0.3时（更宽松）：
模型输出：0.3  0.5  0.7
预测标签：1    1    1
          （对正类更激进）
```

**决策门槛的选择**：
- **医学诊断**：使用低门槛（如 0.3），宁可误诊也不漏诊
- **垃圾邮件检测**：使用高门槛（如 0.7），减少误标记正常邮件
- **一般情况**：使用 0.5（平衡正负预测）

**PyTorch 实现**：

```python
import torch

def predict_with_threshold(probs, threshold=0.5):
    """根据门槛进行预测"""
    predictions = (probs >= threshold).long()
    return predictions

# 示例
probs = torch.tensor([[0.2], [0.5], [0.7], [0.9]])

print("门槛=0.5:")
print(predict_with_threshold(probs, 0.5))  # [0, 1, 1, 1]

print("门槛=0.7:")
print(predict_with_threshold(probs, 0.7))  # [0, 0, 0, 1]

print("门槛=0.3:")
print(predict_with_threshold(probs, 0.3))  # [0, 1, 1, 1]
```

## 二分类评估指标

### 混淆矩阵（Confusion Matrix）

```
              预测正类(1)    预测负类(0)
实际正类(1)    TP (真正)      FN (假负)
实际负类(0)    FP (假正)      TN (真负)
```

**定义**：
- **TP（True Positive）**：正确预测正类
- **TN（True Negative）**：正确预测负类
- **FP（False Positive）**：错误预测为正类（第一类错误）
- **FN（False Negative）**：错误预测为负类（第二类错误）

### 常用评估指标

**1. 准确率（Accuracy）**
$$\text{Accuracy} = \frac{TP + TN}{TP + TN + FP + FN}$$
- 含义：正确预测的比例
- 适用：正负样本数量均衡

**2. 精确率（Precision）**
$$\text{Precision} = \frac{TP}{TP + FP}$$
- 含义：预测正类中，实际是正类的比例
- 适用：关心误诊的任务（医学诊断）

**3. 召回率（Recall/敏感性）**
$$\text{Recall} = \frac{TP}{TP + FN}$$
- 含义：实际正类中，被正确预测的比例
- 适用：关心漏诊的任务（病变检测）

**4. F1 分数**
$$F_1 = 2 \times \frac{\text{Precision} \times \text{Recall}}{\text{Precision} + \text{Recall}}$$
- 含义：精确率和召回率的调和平均
- 适用：综合评估模型性能

**5. 受试者工作特征曲线（ROC Curve）**
- X 轴：假正率（FPR）= FP / (FP + TN)
- Y 轴：真正率（TPR）= TP / (TP + FN)
- AUC：ROC 曲线下面积，范围 [0, 1]，越大越好

### Python 实现

```python
from sklearn.metrics import confusion_matrix, accuracy_score, precision_score, recall_score, f1_score, roc_auc_score

# 真实标签和预测标签
y_true = [1, 0, 1, 1, 0, 1, 0, 0]
y_pred = [1, 0, 1, 0, 0, 1, 1, 0]
y_pred_probs = [0.9, 0.2, 0.8, 0.4, 0.1, 0.7, 0.6, 0.3]

# 混淆矩阵
cm = confusion_matrix(y_true, y_pred)
print(f"混淆矩阵:\n{cm}")

# 各项指标
accuracy = accuracy_score(y_true, y_pred)
precision = precision_score(y_true, y_pred)
recall = recall_score(y_true, y_pred)
f1 = f1_score(y_true, y_pred)
auc = roc_auc_score(y_true, y_pred_probs)

print(f"准确率: {accuracy:.4f}")
print(f"精确率: {precision:.4f}")
print(f"召回率: {recall:.4f}")
print(f"F1分数: {f1:.4f}")
print(f"AUC: {auc:.4f}")
```

## 完整的二分类模型示例

```python
import torch
import torch.nn as nn
import torch.optim as optim
from sklearn.metrics import accuracy_score, precision_score, recall_score, f1_score

class BinaryClassifier(nn.Module):
    """二分类模型"""
    def __init__(self, input_size):
        super().__init__()
        self.fc1 = nn.Linear(input_size, 64)
        self.fc2 = nn.Linear(64, 32)
        self.fc3 = nn.Linear(32, 1)
        self.relu = nn.ReLU()
        self.sigmoid = nn.Sigmoid()
    
    def forward(self, x):
        x = self.relu(self.fc1(x))
        x = self.relu(self.fc2(x))
        x = self.fc3(x)
        return x  # 返回 logits，不需要 Sigmoid

# 创建模型
model = BinaryClassifier(input_size=784)
optimizer = optim.Adam(model.parameters(), lr=0.001)
criterion = nn.BCEWithLogitsLoss()  # 自动应用 Sigmoid

# 生成示例数据
X_train = torch.randn(100, 784)
y_train = torch.randint(0, 2, (100, 1)).float()
X_test = torch.randn(20, 784)
y_test = torch.randint(0, 2, (20, 1)).float()

# 训练
print("开始训练...")
for epoch in range(10):
    # 前向传播
    logits = model(X_train)
    
    # 计算损失
    loss = criterion(logits, y_train)
    
    # 反向传播
    optimizer.zero_grad()
    loss.backward()
    optimizer.step()
    
    print(f"Epoch {epoch+1}/10 | 损失: {loss:.4f}")

# 评估
print("\n评估结果:")
model.eval()
with torch.no_grad():
    logits = model(X_test)
    probs = torch.sigmoid(logits)
    preds = (probs >= 0.5).long()
    
    accuracy = accuracy_score(y_test.numpy(), preds.numpy())
    print(f"准确率: {accuracy:.4f}")
```

## 总结

**分类的关键要点**：
1. **二分类**：输出是概率（0-1），使用 Sigmoid 或 BCEWithLogitsLoss
2. **Sigmoid 函数**：将任意实数映射到 [0,1]，适合二分类
3. **损失函数**：BCE Loss 衡量预测概率与真实标签的差异
4. **决策门槛**：通常设为 0.5，可根据应用场景调整
5. **评估指标**：准确率、精确率、召回率、F1、AUC 等
6. **常用数据集**：MNIST（10分类）、CIFAR-10（10分类）
