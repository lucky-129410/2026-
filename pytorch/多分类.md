## 多分类问题

**多分类**是指预测目标有多个类别（≥3个）的问题。
- **二分类**：是或否（2个类别）
- **多分类**：类别1、类别2、类别3...（≥3个类别）
  例如：手写数字识别（0-9共10类）、图像分类（猫、狗、鸟...）、文本分类

**多分类的输出**：神经网络输出每个类别的概率分布，概率之和为 1

---

### SoftMax 分类器

**SoftMax 函数**的作用：将网络输出转换为概率分布

**公式**：
$$\text{SoftMax}(z_i) = \frac{e^{z_i}}{\sum_{j=1}^{n} e^{z_j}}$$

其中：
- $z_i$ 是第 i 个类别的网络输出（logit）
- n 是总类别数
- 输出的每一项都在 (0,1) 之间
- 所有输出的和为 1

**SoftMax 的性质**：
- ✓ 确保和为一：$\sum_{i=1}^{n} \text{SoftMax}(z_i) = 1$
- ✓ 确保每一项为正：$0 < \text{SoftMax}(z_i) < 1$
- ✓ 可微分：支持反向传播梯度计算

**SoftMax 示例**：
```
输入（logits）：z = [2.0, 1.0, 0.1]

计算 e^z：
e^2.0 = 7.39, e^1.0 = 2.72, e^0.1 = 1.10
总和 = 7.39 + 2.72 + 1.10 = 11.21

SoftMax 输出：
P(类别1) = 7.39 / 11.21 ≈ 0.659
P(类别2) = 2.72 / 11.21 ≈ 0.243
P(类别3) = 1.10 / 11.21 ≈ 0.098
和 = 0.659 + 0.243 + 0.098 = 1.0 ✓
```

### 损失函数

**交叉熵损失（Cross Entropy Loss）**是多分类的标准损失函数

**公式**：
$$L = -\sum_{i=1}^{n} y_i \log(\hat{y}_i)$$

其中：
- $y_i$ 是真实标签的一热编码（One-hot encoding），只有正确类别为1，其余为0
- $\hat{y}_i$ 是模型预测的第 i 个类别的概率
- n 是总类别数

**交叉熵的计算**：

```
假设真实标签是类别 2（共3个类别）
真实标签 y = [0, 1, 0]  （one-hot 编码）

模型预测 y_pred = [0.1, 0.8, 0.1]  （SoftMax输出，和为1）

损失计算：
L = -(0 × log(0.1) + 1 × log(0.8) + 0 × log(0.1))
  = -log(0.8)
  = 0.223

损失越小说明预测越准确！
```

**理解交叉熵**：
- 只看正确类别的概率：$-\log(\hat{y}_{true})$
- 当预测概率接近 1 时，-(log 1) = 0，损失最小
- 当预测概率接近 0 时，-(log 0) ≈ ∞，损失很大

### PyTorch 实现

**单个样本的多分类**：
```python
import torch
import torch.nn as nn

# 创建网络
class MultiClassifier(nn.Module):
    def __init__(self, input_size, num_classes):
        super().__init__()
        self.fc = nn.Linear(input_size, num_classes)
    
    def forward(self, x):
        return self.fc(x)

# 模型、数据和标签
model = MultiClassifier(input_size=784, num_classes=10)  # 10分类
x = torch.randn(1, 784)  # 输入数据
y_true = torch.tensor([3])  # 真实标签（类别3）

# 前向传播
logits = model(x)  # shape: [1, 10]

# 计算损失
# 方法1：分开 SoftMax 和 NLLLoss（负对数似然）
softmax = nn.Softmax(dim=1)
nll_loss = nn.NLLLoss()
loss = nll_loss(torch.log(softmax(logits)), y_true)

# 方法2：直接使用 CrossEntropyLoss（推荐！）
criterion = nn.CrossEntropyLoss()  # 内置 SoftMax + NLL
loss = criterion(logits, y_true)

print(f"损失值: {loss.item():.4f}")
```

**批次数据的多分类（实际使用）**：
```python
import torch
import torch.nn as nn

class MultiClassifier(nn.Module):
    def __init__(self, input_size, hidden_size, num_classes):
        super().__init__()
        self.fc1 = nn.Linear(input_size, hidden_size)
        self.relu = nn.ReLU()
        self.fc2 = nn.Linear(hidden_size, num_classes)
    
    def forward(self, x):
        x = self.fc1(x)
        x = self.relu(x)
        x = self.fc2(x)  # 输出 logits （不需要 SoftMax）
        return x

# 创建模型
model = MultiClassifier(input_size=784, hidden_size=128, num_classes=10)
optimizer = torch.optim.Adam(model.parameters(), lr=0.001)
criterion = nn.CrossEntropyLoss()

# 模拟批次数据
batch_size = 32
x_batch = torch.randn(batch_size, 784)  # 一批数据
y_batch = torch.randint(0, 10, (batch_size,))  # 一批标签 [0-9]

# 训练步骤
def train_step():
    # 1. 前向传播
    logits = model(x_batch)  # [batch_size, 10]
    
    # 2. 计算损失
    loss = criterion(logits, y_batch)
    
    # 3. 梯度清零
    optimizer.zero_grad()
    
    # 4. 反向传播
    loss.backward()
    
    # 5. 参数更新
    optimizer.step()
    
    return loss.item()

loss = train_step()
print(f"训练损失: {loss:.4f}")

# 推理与预测
def predict(x):
    model.eval()  # 评估模式
    with torch.no_grad():
        logits = model(x)
        probs = torch.softmax(logits, dim=1)  # 获得概率
        pred_classes = torch.argmax(probs, dim=1)  # 获得预测类别
    return pred_classes, probs

pred_classes, probs = predict(x_batch[:5])
print(f"预测类别: {pred_classes}")
print(f"预测概率: {probs}")
```

**多分类的关键点**：
- 模型输出：logits（未经过 SoftMax 的原始输出）
- 损失函数：`CrossEntropyLoss`（自动应用 SoftMax）
- 推理时：使用 `torch.softmax()` 获得概率，用 `torch.argmax()` 获得最可能的类别
- 标签格式：长整型张量（LongTensor），值为类别索引（0 到 num_classes-1）
